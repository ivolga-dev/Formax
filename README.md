# Glass Music Card

Мини-сайт в стиле glassmorphism:
- кинематографичная поэтапная анимация появления (лоадер → карточка → элементы);
- видео `background.mp4` на фоне;
- карточка с данными и анимацией появления;
- современный mp3-плеер с кнопкой-стрелкой для открытия меню плейлиста;
- стрелка снизу блока данных для открытия/скрытия меню плейлиста.

## Что нужно положить в проект

### 1) Фоновое видео
Положите файл с именем:

- `background.mp4`

в корень проекта (рядом с `index.html`).

### 2) Музыка
Положите все `.mp3` файлы в папку:

- `mp3/`

Названия треков вводить вручную не нужно: плеер пытается автоматически прочитать файлы из папки `mp3/`.

> Если ваш хостинг не отдает список файлов директории `mp3/`, создайте `mp3/playlist.json` в формате:
> ```json
> ["track1.mp3", "track2.mp3", "track3.mp3"]
> ```

### 3) Иконки
Положите изображения в папку `icons/` с **точно такими именами**:

- `avatar-placeholder.png` — аватар
- `telegram.png` — иконка Telegram
- `tiktok.png` — иконка TikTok
- `arrow-right.png` — стрелка открытия плейлиста
- `play.png` — иконка play
- `pause.png` — иконка pause
- `prev.png` — иконка previous
- `next.png` — иконка next

## Запуск

Откройте `index.html` через локальный сервер, например:

```bash
python3 -m http.server 8080
```

После этого зайдите в браузере на `http://localhost:8080`.


Реализация выполнена на чистых `HTML`, `CSS` и `JavaScript` (без библиотек и фреймворков).
